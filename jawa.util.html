class jawa.util.bytecode.StreamDisassembler(io)[source]¶

Bases: object

Disassembles a stream of JVM instructions without caching or otherwise doing anything with them. Once the stream has been consumed, it cannot be re-iterated.

Parses and returns a single opcode and its operands from the stream io. io must support a blocking read() and tell().

util.descriptor Module¶

Methods for parsing standard JVM type descriptors for fields and methods. It does not depend on any other modules.

Note

This module has not yet been optimized for performance and is not in its final form.

exception jawa.util.descriptor.DescriptorError[source]¶

Bases: exceptions.Exception

jawa.util.descriptor.field_descriptor(descriptor)[source]¶

Parses a Field descriptor as described in section 4.3.2 of the JVM specification. Returns the simple type of the field.

jawa.util.descriptor.method_descriptor(descriptor)[source]¶

Parses a Method descriptor as described in section 4.3.3 of the JVM specification. Returns the method argument types and return type.

jawa.util.descriptor.parse_descriptor(descriptor)[source]¶

Uses a tiny state machine to parse JVM descriptors. To get useful wrappers around the results, use jawa.core.descriptor.method_descriptor() or jawa.core.descriptor.field_descriptor().

util.ezip Module¶

class jawa.util.ezip.EditableZipFile(io=None)[source]¶

Bases: object

A light wrapper around the python zipfile.ZipFile that enables in-memory modification.

If io is provided, its contents are loaded. io may be a file-system path or a file-like object.

close(*exc_info)¶

Closes the ZipFile and any underlying objects.

namelist[source]¶

Returns a list of all paths in this ZipFile.

read(path)[source]¶

Returns the contents of path, if it exists, otherwise None.

regex(regex)[source]¶

Returns an iterator over every path that matches regex. For example, to get all files ending in .class that aren’t in a sub-directory:

>>> for path in jf.regex('[^/]+\.class'):
...    print(path)
remove(path)[source]¶

Removes a file from the ZipFile.

>>> print jf.namelist()
['my_file.txt', 'that_other_file.txt', ...]
>>> jf.remove('my_file.txt')
>>> print jf.namelist()
['that_other_file.txt', ...]
save(path)[source]¶

Saves the ZipFile to path, which can be a file-system path or a file-like object.

write(path, data)[source]¶

Writes data to path, overwritting any existing content.
