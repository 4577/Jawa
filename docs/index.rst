Jawa
================================

Jawa is an easy-to-use, lightly-optimized Python package for working with `JVM ClassFiles <https://en.wikipedia.org/wiki/Java_class_file>`_, most commonly associated with `Java <https://en.wikipedia.org/wiki/Java_(programming_language)>`_. It provides tools for reading and creating JAR archives and Classes, as well as JVM bytecode assembly and disassembly.
