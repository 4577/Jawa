Jawa - JVM ClassFile Library¶

Jawa is a pure-python library for examining and disassembling JVM (Java) class files.

Note

This documentation is up to date as of March 13, 2015.

Getting Jawa¶

Jawa is not yet available on PyPi as it is still being developed. That said, breaking changes are rare and installation from Github is very easy. To install:

pip install git+https://github.com/TkTech/Jawa.git
