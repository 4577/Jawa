class jawa.core.constants.ConstantPool[source]¶

Bases: object

Implements a container around the Constant Pool as described in the JVM ClassFile specification.

build_class(class_name)[source]¶

Builds a new jawa.core.constants.ConstantClass with the name class_name in the pool and returns a tuple of the index it was inserted at and the ConstantClass itself.

find(type_=None, f=None)[source]¶

Iterates over the pool, yielding each matching Constant. Calling without any arguments is equivelent to iterating over the pool.

  • type_ must be a subclass of Constant.
  • f must be callable which takes one value (the constant)
find_index(constant)[source]¶

Returns the index for constant in the pool, or None if it is not in the pool.

find_one(*args, **kwargs)[source]¶

Same as find() but returns only the first result, or None if nothing was found.

get(index, default=None)[source]¶

Returns the Constant at index, or default if it does not exist.

insert(constant, index=None)[source]¶

Inserts the value of constant to index, potentially overwritting an existing value. If index is None, the first free index is used. Returns the final index.

Warning

This does not invalidate a Constant whose indexes point to the overwritten value. They will instead resolve to the newly inserted value.

remove(obj_or_index)[source]¶

Removes obj_or_index from the pool. If obj_or_index is a subclass of jawa.core.constants.Constant then it is removed by identity, otherwise it is deleted by index.

jf Module¶

Tools for working with JVM .jar archives, which are simply ZIP archives with some mandatory structure.

Note

If czipfile is available, decompression will be noticably faster.
